 Personal Bio Website (3 Pages)

 Description:
A simple 3-page personal portfolio website built with semantic HTML5. It includes Home, About, and Contact pages with proper navigation, structured content, accessible forms, and images with alt text.


 Features:
- Multi-page website (Home, About, Contact)
- Semantic HTML5 structure
- Working navigation links
- Contact form with fieldsets and inputs
- Accessible images with alt text
- Mailto and tel links for direct contact
- Clean and organized layout
